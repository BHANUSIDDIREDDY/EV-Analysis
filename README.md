# EV-Analysis
EDA and regression analysis on Electric Vehicle dataset using Python
# 🚗 Electric Vehicle Data Analysis

## 📊 Overview

EDA + regression on EV population data to uncover trends and relationships.

## 📁 Dataset

* ~5000 rows, 16 columns
* Key features: Model Year, Electric Range, Vehicle Type, Base MSRP

## ⚙️ Stack

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, SciPy

## 🔍 What I did

* Data cleaning (NA removal, duplicates)
* Outlier detection (Z-score, IQR)
* Visualizations (line, bar, histogram, scatter, heatmap, boxplot, pie)
* Linear regression (Model Year → Electric Range)

## 📈 Insights

* EV adoption increases over time
* BEVs dominate PHEVs
* Weak linear relation: Model Year vs Range

## 🤖 Model Metrics

* R² Score
* Mean Squared Error
